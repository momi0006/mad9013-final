Responsive Layout with Media Queries

We created a mobile-first responsive layout. We used media queries to create different layouts based on the viewport widths. When the viewport size was smallest the layout was all lined up in a single column. When the viewport size is greater than small layout but less than the largest layout, the layout splits into two-column layout. When the screen size is highest, the layout then converts into a four-column layout.

Responsive Form

In this assignment we created an HTML form which contained four input fields - types text, email, url, and password, plus a submit button. As the width of the screen changes, the layout of the form changed. At the smallest size range the fields lined up in a single vertical column. The labels for were hidden in favour of the placeholder text. At the mid-range sizes there were two columns with labels on top of input fields. At the largest screen widths the labels appeared to the left of each of the input fields.
